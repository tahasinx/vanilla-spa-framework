Quick Start

  1. Clone or download this framework.
  2. Run a local server (do not use file://).
  3. Open index.html in your browser.
  4. Define routes in routes/web.js and build features.
# Example local server
python -m http.server 5500

Project Structure

Everything is organized into core framework modules, routes, controllers, and views.

vanilla-spa-framework/
├── index.html
├── framework/core/
│   ├── Router.js
│   ├── Controller.js
│   ├── View.js
│   ├── Api.js
│   └── App.js
├── routes/web.js
├── controllers/
│   ├── HomeController.js
│   └── ApiController.js
└── resources/
    ├── views/
    └── css/app.css

Routing (Unified + Laravel-like)

Define all web and API routes in routes/web.js using AppRouter.

// Page routes
AppRouter.get('/', 'HomeController', 'index');
AppRouter.get('/docs', 'HomeController', 'docs');
AppRouter.get('/contact', 'HomeController', 'contact');

// API routes
AppRouter.get('/api/users', 'ApiController', 'getUsers');
AppRouter.get('/api/inspire', 'ApiController', 'inspire');

Tip: keep routing centralized in one file for predictable maintenance.

Controllers Pattern

Controllers extend Controller, load templates, and send data to views.

class HomeController extends Controller {
  async docs() {
    if (!this.view.templates['docs']) {
      await this.view.loadTemplate('docs', 'resources/views/docs.html');
    }
    this.view.updateElement('#app', 'docs', {
      title: 'Vanilla SPA Framework Documentation'
    });
  }
}
window.HomeController = HomeController;

Template Syntax (Variables, Conditions, Loops)

Variables

<h1>{{ title }}</h1>
<p>{{ user.name }}</p>

Conditionals

@if(user.isAdmin)
  <div class="admin-panel">Admin Content</div>
@else
  <div class="basic-panel">Basic Content</div>
@endif

Foreach Loop

@foreach(users as user)
  <div class="user-card">
    <h4>{{ user.name }}</h4>
    <p>{{ user.email }}</p>
  </div>
@endforeach

For Loop

@for(let i=1;i<=5;i++)
  <div>Index: ${i}</div>
@endfor

Use ${i} for loop variables inside @for blocks.

API Integration

Use one API helper for internal controller routes and external network APIs.

// In controller methods
const users = await this.api.get('/api/users');
const created = await this.api.post('/api/users', { name: 'John' });

// DOM helper methods
this.api.updateElement('#users', '/api/users');
this.api.renderTemplate('#users', 'user-list', '/api/users');

CORS Notes

External APIs must allow CORS. Frontend JS cannot bypass CORS restrictions by itself.

// Development-only proxy example
https://corsproxy.io/?https://example.com/api

Form Handling

Add data attributes and let the framework submit/dispatch events automatically.

<form data-action="/api/users" data-method="POST" data-target="#result">
  <input type="text" name="name" required />
  <input type="email" name="email" required />
  <button type="submit">Submit</button>
</form>
<div id="result"></div>

You can listen to form-success and form-error events on the form.

Configuration

Set API base URL and debug mode from your startup script.

App.config({
  apiUrl: 'https://api.example.com',
  debug: true
});
